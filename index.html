Profile Track

Digital Footprint Discovery Tool

Description

Profile Track is a comprehensive system that allows users to discover websites or platforms where their email address or username might have been registered or publicly exposed. This cross-platform solution consists of a Flutter mobile application that works in conjunction with a powerful Spring Boot backend service.

Profile Track, e-posta adresinizin veya kullanıcı adınızın kayıtlı olabileceği web sitelerini ve platformları keşfetmenizi sağlayan kapsamlı bir sistemdir. Bu çapraz platform çözümü, güçlü bir Spring Boot arka uç hizmetiyle birlikte çalışan bir Flutter mobil uygulamasından oluşur.

Domain Track

Domain Intelligence Tool

Description

Domain Track is a sophisticated cross-platform mobile application built with Flutter, designed to provide comprehensive domain intelligence services. The application allows users to query domain availability across multiple TLDs and fetch detailed WHOIS registration information including registrar details, creation and expiration dates, and contact emails.

Domain Track, kapsamlı alan adı istihbarat hizmetleri sağlamak için Flutter ile geliştirilmiş sofistike bir çapraz platform mobil uygulamadır. Uygulama, kullanıcıların birden fazla TLD'de alan adı kullanılabilirliğini sorgulamasına ve kayıt detayları, oluşturma ve sona erme tarihleri ile iletişim e-postaları dahil olmak üzere detaylı WHOIS kayıt bilgilerini getirmesine olanak tanır.
